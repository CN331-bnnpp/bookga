# Bookga Project

## Description
web application for book work shift for employee and shift schedule management for small business owners.

## Features
- Book work shift
- Shift schedule management
- User management
- User authentication
- Request management
- Statistics report

## Tech Stack
- Frontend: html, css
- Backend: python, django
- Database: sqlite3
